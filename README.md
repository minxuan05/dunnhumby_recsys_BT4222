# BT4222 Group 1: Grocery Binary Classification Recommendation Model

## Project Overview
This project implements a Binary Classification Recommendation System designed to predict user purchase behavior. By leveraging historical transaction data and demographic metadata, the model identifies the most relevant items for a given user, facilitating a personalised shopping experience that maximizes transaction value and user engagement.

## Dataset
We utilised the "Complete Journey" provided by Dunnhumby , which captures the household-level shopping behaviour of 2,500 shoppers over two years.   

https://www.kaggle.com/datasets/frtgnn/dunnhumby-the-complete-journey?select=transaction_data.csv

Key Files Used:
hh_demographic.csv: Household metadata including income, family size, and age.
transaction_data.csv: Transaction records. 
product.csv: Item-level metadata such as department, brand, and commodity description.
